Your Chrome user and Google Music user don't seem to match, which is required due to restrictions in Google's apis. To fix this:

  • If you sign into Google Music under a different user than Chrome, create a new Chrome profile matching your Google Music user, turn on sync, and then install the extension to that profile instead.
  • If your users already match, check that Chrome syncing is enabled and working, then restart Chrome.

If you're still seeing this error after trying both approaches, feel free to email me for help: simon@simonmweber.com
